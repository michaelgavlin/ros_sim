To use my customized robot copy files:
	sudo cp turtlebot3_burger_michael.urdf.xacro /opt/ros/noetic/share/turtlebot3_description/urdf
	sudo cp turtlebot3_burger_michael.gazebo.xacro /opt/ros/noetic/share/turtlebot3_description/urdf

And launch file:
	cp turtlebot3_empty_world_michael.launch ~/catkin_ws/src/turtlebot3_simulations/turtlebot3_gazebo/launch
